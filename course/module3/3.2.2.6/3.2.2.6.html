Modo avanzado de UEFI

La pantalla del menú principal de modo avanzado de ASUS aparece cuando ingresa el modo avanzado. Como se muestra en la Figura 1, este modo brinda opciones para la configuración avanzada de UEFI.

Nota: Para regresar al modo EZ, presione F7.

Estos son los elementos que se encuentran en el menú superior de la pantalla:

  • Mis favoritos: (Figura 2) se utiliza para proporcionar acceso rápido a las opciones de configuración usadas con frecuencia. Presione F3 o haga clic en Mis favoritos (F3) para abrir la pantalla del mapa de árbol de la configuración que se muestra en la Figura 3.

  • Principal: (Figura 4) Se utiliza para mostrar información acerca de la computadora y cambiar la configuración básica. Le permite configurar la fecha, la hora, el idioma y la configuración de seguridad.

  • Ai Tweaker: (Figura 5) Utilizado para cambiar la configuración de overclocking de la CPU y las opciones de configuración de funcionamiento de memoria. Sea siempre cuidadoso al cambiar estas opciones de configuración. Las opciones de configuración incorrectas pueden provocar que el sistema no funcione correctamente.

  • Avanzado (Figura 6) Utilizado para cambiar las opciones de configuración de la CPU y otros dispositivos. Sea siempre cuidadoso al cambiar estas opciones de configuración. Las opciones de configuración incorrectas pueden provocar que el sistema no funcione correctamente.

  • Monitor: (Figura 7) Utilizado para mostrar las temperaturas, estado de energía, y cambiar las opciones de configuración de velocidad del ventilador.

  • Arranque: (Figura 8) Utilizado para cambiar las opciones de arranque.

  • Herramienta: (Figura 9) Utilizado para configurar las opciones para las funciones especiales.

  • Salir: (Figura 10) Utilizado para seleccionar las opciones de salida y cargar las opciones de configuración predeterminadas.
