Otros métodos de instalación

Una instalación estándar de Windows es suficiente para la mayoría de las computadoras utilizadas en un entorno doméstico o de oficinas pequeñas pero hay casos en que se requiere un proceso de instalación personalizada.

Por ejemplo, un departamento de soporte de TI; los técnicos en estos entornos deben instalar cientos de sistemas Windows. No es viable la realización de muchas instalaciones en el modo estándar.

Una instalación estándar se realiza mediante los programas de instalación (DVD o unidad USB) que Microsoft proporciona y es un proceso interactivo; el instalador solicita al usuario para las opciones de configuración como zona horaria y lenguaje del sistema.

La instalación personalizada de Windows puede ahorrar tiempo y proporcionar una configuración coherente en todas las computadoras dentro de una organización extensa. Una técnica popular para instalar Windows en muchas computadoras es realizar la instalación en una computadora y usarla como instalación de referencia. Cuando la instalación se ha completado, se crea una imagen. Una imagen es un archivo que contiene todos los datos de una partición.

Cuando la imagen está lista, los técnicos pueden realizar una instalación mucho más corta simplemente replicando e implementando la imagen a todas las computadoras de la organización. Si la nueva instalación requiere algunos ajustes, pueden realizarse rápidamente después de que se implementa la imagen.

Una imagen de un sistema se crea según los siguientes pasos:

Paso 1. Realizar la instalación completa de Windows en una computadora. Esta computadora debe ser tan similar como las computadoras que recibirán la posterior instalación para evitar problemas de incompatibilidad del controlador.

Paso 2. Utilizando una herramienta de software como imageX, cree una imagen de la instalación. La figura muestra la herramienta de línea de comando imageX.

Paso 3. El resultado debe ser un archivo de imagen grande que contenga una copia de la instalación del sistema operativo completa actualmente presente en el sistema de referencia.

En Windows Vista y anteriores, la instalación estándar se realizó según los archivos individuales incluidos en el DVD de instalación. Actualmente, al instalar Windows 7 o 8.x, el proceso de instalación se basa en un archivo de imagen y aplicar una imagen en el disco duro del DVD de instalación: install.wim.

Windows tiene distintos tipos de instalaciones personalizadas:

  • Instalación de la red: Esto incluye la instalación del entorno de ejecución de prearranque, la instalación, la instalación desatendida y la instalación remota.

  • Instalación de partición interna basada en imagen.

  • Otros tipos de instalaciones personalizadas: Esto incluye las opciones de inicio avanzadas de Windows, actualizar su PC (Windows 8.x solamente), la restauración del sistema, actualización, reparación de instalación, instalación de la red remota, recuperación de partición y actualización/restauración.
