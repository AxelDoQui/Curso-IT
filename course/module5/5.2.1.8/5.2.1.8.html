Creación de cuentas

Cuando los usuarios intentan registrarse a un dispositivo o acceder a los recursos del sistema, Windows utiliza el proceso de autenticación para verificar la identidad de los usuarios. La autenticación tiene lugar cuando el usuario introduce un nombre de usuario y una contraseña para acceder a una cuenta de usuario. Los sistemas operativos Windows utilizan la autenticación de inicio de sesión único (SSO, Single-Sign On), que les permite a los usuarios iniciar sesión una vez para acceder a todas las características del sistema, en lugar de pedirles que inicien sesión cada vez que necesiten acceder a un recurso específico.

Las cuentas de usuario permiten que varios usuarios compartan una sola computadora usando sus propios archivos y opciones de configuración. Windows 8.1, Windows 7 y Windows Vista tienen tres tipos de cuentas de usuario: Administrador, estándar, e invitado. Cada uno de estos tipos de cuentas le proporciona al usuario un nivel de control distinto sobre los recursos del sistema.

En Windows 8.1, la cuenta creada durante el proceso de instalación tiene privilegios de administrador. Un usuario con privilegios de administrador puede realizar modificaciones que afecten a todos los usuarios de la PC, como alterar la configuración de seguridad o instalar software para todos los usuarios. Las cuentas con privilegios de administrador se deben utilizar solo para administrar una PC y no se debe usar regularmente, ya que, al usar la cuenta de administrador, se pueden realizar modificaciones drásticas que afectan a todos los usuarios. Además, los atacantes buscan las cuentas de administrador debido al poder que tienen estas cuentas. Por esta razón, se recomienda crear una cuenta de usuario estándar para uso habitual.

Las cuentas de usuario estándar pueden ser creadas en cualquier momento por el administrador. Una cuenta de usuario estándar puede utilizar la mayor parte de la funcionalidad de una computadora; sin embargo, el usuario no puede realizar cambios que afecten a otros usuarios o a la seguridad de la computadora. Por ejemplo, un usuario estándar no puede instalar una impresora.

Los usuarios que no poseen una cuenta de usuario en la computadora pueden utilizar una cuenta de invitado. Las cuentas de invitado tienen permisos limitados y las debe activar un administrador. La aplicación del panel de control de administración de cuentas se utiliza para administrar las cuentas de usuario. El panel de control es una ubicación de Windows que contiene muchas herramientas para manipular la configuración de Windows. Existen muchas maneras de acceder al panel de control en cada versión de Windows.

Para acceder al Panel de control de Windows 7 y Vista, haga clic en el botón Inicio ubicado en el lado izquierdo de la barra de tareas en la parte inferior del escritorio. Se mostrará el menú Inicio. Haga clic en Panel de control en el menú inicio. También puede hacer clic en el botón Inicio y escribir control en el cuadro Buscar programas y archivos y presionar Intro.

La interfaz de Windows 8 es muy diferente de la de Windows 7 y Windows Vista. Para acceder al panel de control del escritorio de Windows 8, coloque el cursor en la esquina superior derecha de escritorio y espere que aparezcan los accesos. Haga clic en el acceso Buscar y escriba control en el cuadro de búsqueda y presione Intro. En una pantalla táctil, también puede deslizar el dedo del lado derecho de la pantalla para mostrar los accesos.

Windows 8 no viene con un menú de inicio de manera predeterminada. Windows 8 utiliza una pantalla de inicio en su lugar. Para acceder a la pantalla de inicio, primero acceda a los accesos y haga clic en el icono Inicio. Para tener acceso al panel de control de la pantalla de inicio, escriba control y presione Intro. También puede acceder a la pantalla de inicio en Windows 8.1 haciendo clic en el botón Inicio ubicado en el lado izquierdo de la barra de tareas en la parte inferior del escritorio.

El panel de control se puede ver de distintas maneras, según la versión de Windows que está utilizando. En este curso, las rutas a las herramientas del panel de control asumirán que la vista esté configurada en iconos grandes o pequeñas. Esto se puede resolver eligiendo iconos grandes o iconos pequeños en el menú desplegable Ver por: en el panel de control. En Windows Vista, los iconos pueden verse haciendo clic en Vista clásica en el panel de control.

En este curso, las rutas a las herramientas del panel de control comienzan con Panel de control.

Para administrar una cuenta de usuario en Windows 8.1 y Windows 8, utilice la siguiente ruta:

Panel de control > Cuentas de usuario > Administrar cuentas

Para crear o eliminar una cuenta de usuario en Windows 7 y Windows Vista, utilice la siguiente ruta:

Inicio > Panel de control > Cuentas de usuario > Agregar o quitar cuentas de usuario

La figura muestra la ventana de administración de cuentas de Windows 8.1.
