Clonación de discos

La instalación de un sistema operativo en una sola PC lleva tiempo. Imagínese el tiempo que llevaría instalar sistemas operativos en varias PC, una por una. Para simplificar esta actividad, los administradores eligen generalmente una computadora para actuar como sistema básico y pasar por el proceso regular de instalación del sistema operativo. Después de que el sistema operativo esté instalado en la computadora base, se utiliza un programa específico para copiar toda la información en su disco, sector por sector, a otro disco. Este nuevo disco, generalmente un dispositivo externo, ahora contiene un sistema operativo completamente implementado y se puede utilizar para implementar rápidamente una copia instalada nueva del sistema operativo base. Debido a que el disco de destino ahora contiene una asignación de sector a sector de disco original, el contenido del disco de destino es una imagen del disco original.

Si una configuración no deseada se incluye accidentalmente durante la instalación base, un administrador puede utilizar la herramienta de preparación del sistema de Microsoft (Sysprep) para eliminarla antes de crear la imagen final. Sysprep se puede utilizar para instalar y configurar el mismo sistema operativo en varias computadoras. Sysprep prepara al sistema operativo con diferentes configuraciones de hardware. Con Sysprep los técnicos pueden instalar el sistema operativo, completar los últimos pasos de configuración e instalar aplicaciones rápidamente.
