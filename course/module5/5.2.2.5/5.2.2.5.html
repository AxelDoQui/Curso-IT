Opciones de recuperación del sistema

Cuando se produce un error del sistema, los usuarios pueden emplear las siguientes herramientas de recuperación:

  • Opciones de inicio avanzadas de Windows (Windows 8.x)

  • Opciones de recuperación del sistema (Windows 7 y Windows Vista)

  • Partición de recuperación de fábrica

Opciones de inicio avanzadas de Windows 8.x

Las opciones de inicio avanzadas son un conjunto de herramientas de Windows 8.x que permite a los usuarios resolver problemas, recuperar o restaurar un sistema operativo cuando no puede arrancar. Las opciones avanzadas son parte del entorno de recuperación de Windows (WinRE). WinRE es una plataforma de recuperación basada en el Entorno de preinstalación de Windows (WinPE, Windows Preinstallation Environment). Para acceder a herramientas avanzadas, presione la tecla F8 durante el inicio para interrumpirlo. Desde la ventana elegir una opción (Figura 1), seleccione solución de problemas. Desde la ventana de problemas (Figura 2), elija las opciones avanzadas. La Figura 3 muestra la ventana Opciones avanzadas con las siguientes herramientas:

  • Restauración del sistema: Esta herramienta restaura una computadora a un punto de restauración anterior. Esto es lo mismo que al utilizar la herramienta restaurar sistema en Windows.

  • Recuperación de imagen del sistema: Esta herramienta restaura la computadora con un archivo de imagen del sistema.

  • Reparación automática: Esta herramienta examina el sistema e intenta reparar automáticamente los problemas que pueden evitar que Windows arranque correctamente.

  • Símbolo del sistema: Abra el símbolo del sistema del entorno de recuperación que otorga acceso a varias herramientas para la solución de problemas de la línea de comandos.

  • Configuración de firmware de UEFI: Esta herramienta se muestra solo si la computadora admite UEFI. Utilice esta herramienta para cambiar la configuración del UEFI de la computadora.

  • Configuración de inicio de Windows: La opción Configuración de inicio le permite habilitar el modo seguro. También puede deshabilitar el reinicio automático después de una falla y permitirle ver el mensaje de error que aparece en la pantalla azul.

Windows 7 y Windows Vista

Similar a opciones avanzadas de Windows 8.x, las opciones de recuperación del sistema son un conjunto de herramientas para la solución de problemas para Windows 7 y Windows Vista. Como se muestra en la Figura 4, las opciones de recuperación del sistema también son parte del entorno de recuperación de Windows.

Como en Windows 8.x, en Windows 7 y Windows Vista, se puede acceder al entorno de recuperación de Windows presionando y sujetando la tecla F8 al iniciar la computadora. Una vez que aparece la pantalla de opciones de arranque avanzadas, resalte Reparar el equipo y presione Intro para acceder a Opciones de recuperación del sistema. A continuación, puede utilizar las herramientas de recuperación del sistema para reparar errores que impiden el arranque del sistema. En el menú Opciones de recuperación del sistema, se encuentran disponibles las siguientes herramientas:

  • Reparación de inicio: Esta herramienta examina el disco duro para detectar problemas y corrige automáticamente archivos de sistema faltantes o dañados que impiden que Windows se inicie.

  • Restaurar sistema: Esta herramienta utiliza puntos de restauración para restaurar los archivos de sistema de Windows a un momento anterior.

  • Recuperación de imagen del sistema: Esta herramienta restaura una imagen creada anteriormente en el disco.

  • Diagnóstico de memoria de Windows: Esta herramienta examina la memoria de la computadora para detectar un funcionamiento incorrecto y para diagnosticar problemas.

  • Símbolo del sistema: Esta herramienta abre una ventana del símbolo del sistema donde se puede utilizar la herramienta bootrec.exe para reparar y resolver problemas de arranque de Windows.

Cuando utilice un disco de recuperación, asegúrese de que este utilice la misma arquitectura que el sistema operativo que se desea recuperar. Por ejemplo, si la PC ejecuta una versión de Windows 7 de 64 bits, el disco de recuperación debe utilizar una arquitectura de 64 bits.
