Administración de aplicaciones y carpetas

Aplicaciones

Las aplicaciones y carpetas de iOS funcionan de manera similar a las del OS Android. Todas las aplicaciones instaladas en el dispositivo se ubican en las pantallas de inicio en lugar de tener un botón denominado Todas las aplicaciones. Las aplicaciones se pueden desinstalar del dispositivo y pueden volver a instalarse por medio de iTunes.

Cada pantalla de inicio tiene una cuadrícula en la que se pueden ubicar las aplicaciones. Para mover una aplicación, siga estos pasos:

Paso 1: Toque la aplicación y manténgala presionada hasta que se mueva.

Paso 2: Arrástrela a un área vacía de alguna de las pantallas de inicio.

Paso 3: Suelte la aplicación.

Paso 4: Mueva las aplicaciones adicionales.

Paso 5: Presione el botón de inicio para guardar los cambios.

Para eliminar una aplicación de un dispositivo iOS, siga estos pasos:

Paso 1: Toque la aplicación y manténgala presionada hasta que se mueva.

Paso 2: Toque el icono X de la aplicación.

Paso 3: Elimine cualquier aplicación adicional.

Paso 4: Presione el botón de inicio para guardar los cambios.

Barra de multitarea

iOS permite que se ejecuten varias aplicaciones al mismo tiempo. Mientras trabaja con una aplicación, se pueden ejecutar otras en segundo plano. La barra de multitarea se utiliza para pasar rápidamente de una aplicación a otra que se acaba de utilizar, para cerrar aplicaciones en uso y para acceder a las configuraciones de uso frecuente. Para abrir la barra de multitarea, como se muestra en la Figura 1, haga doble clic en el botón de inicio. Si bien la barra de tareas es visible, iOS también mostrará los contactos favoritos y recientes en la parte superior de la pantalla. Con la barra de multitarea, se pueden realizar las siguientes acciones:

  • Tocar una aplicación para abrirla.

  • Deslice la barra de multitarea hacia la derecha o hacia la izquierda para ver más aplicaciones.

  • Pulse cualquier contacto favorito o reciente en la parte superior de la pantalla para acceder a él.

  • Deslice una aplicación para forzar el cierre de la aplicación.

Carpetas

También es posible crear carpetas en los dispositivos iOS y mantenerlas organizadas. Para crear una carpeta, siga estos pasos:

Paso 1: Toque una aplicación de una pantalla de inicio y manténgala presionada hasta que se mueva.

Paso 2: Arrastre la aplicación sobre otra que desee colocar en la misma carpeta.

Paso 3: Suelte la aplicación.

Paso 4: Agregue más aplicaciones a la carpeta.

Paso 5: Toque el botón de inicio para guardar los cambios.

Para abrir una carpeta, tóquela. Para abrir una aplicación de la carpeta, tóquela. Para cambiar el nombre de una carpeta, tóquela, luego toque el nombre de la carpeta y escriba el nuevo nombre que desee asignarle. Para cerrar una carpeta, toque el botón de inicio o simplemente toque fuera de la carpeta. En las pantallas de inicio, las carpetas se pueden mover del mismo modo que las aplicaciones.

Para eliminar una aplicación de una carpeta, siga estos pasos:

Paso 1: Abra la carpeta.

Paso 2: Toque la aplicación que desea eliminar y manténgala presionada.

Paso 3: Arrastre la aplicación a un espacio vacío de la pantalla de inicio.

Paso 4: Suelte la aplicación.

Para eliminar una carpeta, elimine todas las aplicaciones de la carpeta.

Muchas aplicaciones de los dispositivos iOS pueden mostrar una notificación de alerta. Una notificación de alerta aparece como un icono pequeño sobre una aplicación, como se muestra en la Figura 2. Este icono es un número que indica la cantidad de alertas de la aplicación. Este número puede tratarse de la cantidad de llamadas perdidas, de mensajes de texto recibidos o de actualizaciones disponibles. Si la notificación de alerta muestra un signo de admiración, significa que hay un problema con la aplicación. Las notificaciones de alerta correspondientes a las aplicaciones que estén dentro de una carpeta aparecen sobre la carpeta. Toque la aplicación para revisar las alertas.
