Descripción general de los bloqueos de contraseña

Los smartphones, las tablet PC y demás dispositivos móviles contienen datos confidenciales. Si se pierde un dispositivo móvil, cualquier persona que lo encuentre puede acceder a los contactos, los mensajes de texto y las cuentas web. Un método para ayudar a evitar el robo de información privada de los dispositivos móviles es utilizar un bloqueo de contraseña. Las contraseñas bloquean los dispositivos y los dejan en estado de ahorro de energía. El bloqueo también puede demorarse para que comience en un tiempo determinado una vez que el dispositivo se encuentre en estado de ahorro de energía. Un método frecuente para poner un dispositivo móvil en estado de suspensión es presionar rápidamente el botón de encendido principal. También se puede configurar el dispositivo para que entre en estado de suspensión después de un tiempo determinado.

Se encuentran disponibles muchos tipos de bloqueos de contraseña, como se muestra en la Figura 1. Algunos tipos de bloqueos de contraseña son más difíciles de adivinar que otros. La contraseña debe introducirse al encender el dispositivo o al reanudar la actividad tras encontrarse en estado de ahorro de energía. Los siguientes son algunos de tipos comunes de bloqueo de contraseña:

  • Ninguno: Elimina cualquier tipo de bloqueo existente de contraseñas para ese dispositivo.

  • Deslizamiento: El usuario desliza un icono, como un candado o una flecha, para desbloquear el dispositivo. Esta es la opción menos segura.

  • Patrón: Bloquea el dispositivo cuando el usuario desliza el dedo por la pantalla siguiendo un patrón específico. Para desbloquear el dispositivo, se debe repetir el mismo patrón en la pantalla.

  • PIN: Utiliza un número de identificación personal (PIN) para proteger el dispositivo. Un PIN es una serie de números. Al introducir el PIN de manera correcta, el dispositivo se desbloquea.

  • Contraseña: Utiliza una contraseña para proteger el dispositivo. Esta es la opción menos conveniente, en especial si la palabra es complicada o larga, pero puede ser muy segura.

  • Código de acceso simple: Solo para dispositivos iOS. Cuando se activa esta opción, el código de acceso debe ser un número de cuatro dígitos. Cuando se desactiva, se pueden utilizar contraseñas más complejas con caracteres, números y símbolos.

  • ID táctil: Dispositivos iOS únicamente. A partir del iPhone 5S, también es posible liberar un dispositivo iOS en utilizando las huellas digitales del usuario. El botón de inicio en los dispositivos seleccionados de iPhone contiene un escáner de huellas digitales que se puede utilizar para desbloquear el dispositivo, para autorizar pagos y para acceder a aplicaciones. Aunque Google ha anunciado la función similar, el desbloqueo de huellas digitales no era muy popular entre los dispositivos Android cuando este capítulo fue escrito.

Para establecer una contraseña en un dispositivo Android, utilice la siguiente ruta:

Configuración> Seguridad > Bloqueo de pantalla > Elija el tipo de bloqueo de contraseña de la lista que desea utilizar y establezca el resto de la configuración de seguridad de la pantalla.

Una vez establecida, la contraseña debe introducirse al encender el dispositivo o al reanudar la actividad tras encontrarse en estado de ahorro de energía.

Android 5.0 (Lollipop) permite un conjunto adicional de opciones liberadas, como se muestra en la Figura 2. Ubicado bajo función inteligente de bloqueo, las opciones extendidas de liberación de Android son:

  • Dispositivos de confianza: Esta función desbloquea el teléfono Android o una tablet cuando se empareja con un dispositivo Bluetooth específico o con la etiqueta NFC de elección del usuario. Por ejemplo, es conveniente poder mantener un teléfono Android o una tablet desbloqueados cuando se emparejan al Bluetooth de un automóvil.

  • Lugares de confianza: Mediante el chip de GPS incorporado en el dispositivo, Android permite al usuario definir la casa y los lugares de trabajo. No será necesario volver a desbloquear el dispositivo cuando se encuentre en una de esas ubicaciones.

  • Cara de confianza: Cuando está activada, la función de cara de confianza permite que Android se libere según una imagen previamente registrada de la cara del usuario.

  • Voz de confianza: Esta función depende del espectrograma de voz único del usuario para desbloquear el dispositivo Android cuando un usuario registrado dice “Okay, Google”.

  • Detección en el cuerpo: Esta función utilizará el acelerómetro del dispositivo para detectar si el usuario tiene el dispositivo en su poder. Mediante la detección de un patrón de movimiento específico, Android supone que el dispositivo se encuentra en las manos, en el bolsillo o en el bolso del usuario y que no será necesario desbloquearlos hasta que sea establecido por el usuario.

Para establecer un código de acceso en un dispositivo Android, utilice la siguiente ruta de acceso:

Configuración > Seguridad > Bloqueo inteligente > Elija el tipo de bloqueo que desea utilizar de la lista y establezca los detalles restantes en función de la opción elegida.

Para establecer una contraseña en un dispositivo iOS, utilice la siguiente ruta:

Configuración > ID táctil y código de acceso > Activar el código de acceso. Introduzca un número de cuatro dígitos, como se muestra en la Figura 3. Ingrese el mismo número otra vez para verificarlo.

Para utilizar el ID táctil, el usuario debe registrar al menos una huella digital, y puede registrar hasta cinco huellas digitales. La configuración y el comportamiento del ID táctil se pueden configurar también en la pantalla descrita anteriormente.

Nota: Para mejorar la seguridad, algunos servicios obligarán al usuario a crear un bloqueo de contraseña para poder habilitarse. Las cuentas de VPN y Microsoft Exchange son dos ejemplos de estos servicios.
