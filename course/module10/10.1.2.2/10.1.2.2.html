Administración de aplicaciones, widgets y carpetas

Aplicaciones

Cada pantalla de inicio está configurada con una cuadrícula en la cual se pueden ubicar las aplicaciones. Para mover una aplicación, siga estos pasos:

Paso 1: Toque la aplicación y manténgala presionada.

Paso 2: Arrástrela a un área vacía de alguna de las pantallas de inicio.

Paso 3: Suelte la aplicación.

Para eliminar una aplicación de una pantalla de inicio, siga estos pasos:

Paso 1: Toque la aplicación y manténgala presionada.

Paso 2: Arrástrela a X Eliminar en la parte superior de la pantalla.

Paso 3: Suelte la aplicación.

Para ejecutar una aplicación, tóquela. Una vez que se ejecuta la aplicación, hay opciones que pueden configurarse tocando el botón de menú.

En general, hay tres formas de cerrar una aplicación:

  • Tocar el botón Atrás de manera continua para llegar a la pantalla de inicio. El programa suele mostrar una solicitud para salir de la aplicación.

  • Toque el botón de Inicio.

  • Tocar la opción Salir en el menú de la aplicación.

Widgets

Los widgets son programas (o partes de programas) que muestran información directamente en la pantalla de inicio. Por ejemplo, un widget del clima puede ubicarse en la pantalla de inicio para mostrar las condiciones climáticas. En general, los widgets se pueden tocar para iniciar una aplicación relacionada. En el caso del ejemplo anterior, al tocar el widget, se abriría una aplicación del clima en modo pantalla completa para mostrar más información y detalles sobre las condiciones climáticas. Los widgets son útiles, porque brindan un acceso rápido a la información más consultada o a las funciones de uso más frecuente. Estos son ejemplos de los widgets más comunes:

  • Reloj: Muestra una versión de mayor tamaño de un reloj personalizable.

  • Clima: Muestra las condiciones meteorológicas actuales en uno o más lugares.

  • Wi-Fi activada/desactivada: Permite que el usuario active y desactive rápidamente la conexión wifi sin necesidad de ir al menú de configuración.

  • Control de encendido: Muestra varios widgets, como wifi activada/desactivada, Bluetooth y vibración.

Existen muchos otros tipos de widgets que se pueden utilizar para personalizar las pantallas de Android. Consulte los documentos de los programas o Google Play para determinar qué aplicaciones tienen widgets. Para agregar un widget a la pantalla de inicio, siga estos pasos:

Paso 1: Toque y mantenga presionada un área vacía en la pantalla de inicio.

Paso 2: Toque los widgets.

Paso 3: Identifique el widget deseado de la lista.

Paso 4: Toque y mantenga presionado el widget deseado y arrástrelo a la pantalla de inicio.

Carpetas

En algunos dispositivos móviles, se pueden agrupar varias aplicaciones en carpetas para mantenerlas organizadas. Si las carpetas no están disponibles, es posible instalar una aplicación que proporcione esta característica. Las aplicaciones se pueden agrupar del modo en que se desee.

Para crear una carpeta en un dispositivo Android, siga estos pasos:

Paso 1: Toque una aplicación de una pantalla de inicio y manténganla presionada.

Paso 2: Arrastre la aplicación sobre otra que desee colocar en la misma carpeta.

Paso 3: Suelte la aplicación.

Para abrir una carpeta, tóquela. Para abrir una aplicación de la carpeta, tóquela. Para cambiar el nombre de una carpeta, tóquela, luego toque el nombre de la carpeta y escriba el nuevo nombre que desee asignarle. Para cerrar una carpeta, toque el botón Atrás o Inicio, o simplemente toque fuera de la carpeta. En las pantallas de inicio, las carpetas se pueden mover del mismo modo que las aplicaciones.

Para eliminar una aplicación de una carpeta, siga estos pasos:

Paso 1: Abra la carpeta.

Paso 2: Toque la aplicación que desea eliminar y manténgala presionada.

Paso 3: Arrastre la aplicación a un espacio vacío de la pantalla de inicio.

Paso 4: Suelte la aplicación.

En los dispositivos Android, cuando una carpeta tiene dos aplicaciones y se elimina una de ellas, la aplicación que queda reemplaza a la carpeta (la cual desaparece) en la pantalla de inicio.

Icono Todas las aplicaciones

El icono Todas las aplicaciones abre la pantalla Todas las aplicaciones. Esta opción muestra todas las aplicaciones instaladas en el dispositivo. Las siguientes son algunas de las tareas más frecuentes que se pueden realizar en la pantalla Todas las aplicaciones:

  • Iniciar una aplicación: Toque una aplicación para iniciarla.

  • Ubicar aplicaciones en una pantalla de inicio: Toque una aplicación y manténgala presionada. Se muestra la pantalla de inicio. Suelte la aplicación en cualquier espacio libre de una pantalla de inicio.

  • Desinstalar aplicaciones: Toque una aplicación y manténgala presionada. Arrástrela hasta la papelera o el icono X.

  • Acceder a Play Store: Toque el icono menú y luego Google Play.
