Introducción a sistemas operativos Linux y OS X

Unix es un sistema operativo no exclusivo basado en el lenguaje de programación C y el comando interfaz del usuario. Algunos sistemas operativos de escritorio populares se basan en UNIX, como Linux, OS X, iOS y Android.

Linux

Los sistemas operativos Linux se utilizan en prácticamente cada plataforma, incluidos los sistemas incorporados, los dispositivos ponibles, los relojes inteligentes, los teléfonos móviles, las netbooks, las computadoras, los servidores y las súper computadoras. Aunque Linux está obteniendo una base de usuarios más grande, Android, una versión modificada de Linux, será responsable de la expansión del sistema operativo en el mercado de consumo.

OS X (anteriormente conocido como Mac OS X)

El sistema operativo para las computadoras Macintosh es OS X. Se conocía anteriormente como Mac OS X. El OS X se optimiza para el hardware de la computadora Macintosh y puede trabajar perfectamente con otros dispositivos de Apple, como iPhones.

La Figura 2 muestra el escritorio del OS X 10.10 Yosemite

Nota: Este capítulo se centra en Ubuntu Linux 14.04 LTS (Trusty Tahr) y el OS X 10.10 (Yosemite) de Apple.

Una herramienta especial integrada al OS X permite algunas configuraciones. La herramienta se denomina Netboot y puede arrancar varias máquinas OS X de manera remota. Una vez que las máquinas se reinician, cualquier contenido de la sesión desaparece. Esto puede usarse en una configuración como un salón de clases para restablecer las computadoras al final de una clase o del día. La herramienta también puede configurar una instalación del SO o un programa en varias computadoras al mismo tiempo. Esto sería útil en una empresa corporativa para realizar una imagen o actualizar todas las computadoras.
