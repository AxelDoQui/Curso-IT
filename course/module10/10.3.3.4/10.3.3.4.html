Configuración de correo electrónico en iOS

Los dispositivos iOS se entregan con una aplicación común de correo que puede administrar las cuentas de correo electrónico simultáneamente. La aplicación de correo también admite diferentes tipos de cuenta de correo electrónico, incluidos iCloud, Yahoo, Gmail, Outlook y Microsoft Exchange.

Para configurar un dispositivo iOS, es necesario contar con una ID de Apple. Las ID de Apple se utilizan para acceder a la Apps Store de Apple, la tienda de iTunes e iCloud. iCloud proporciona correo electrónico y la capacidad de almacenar contenido en servidores remotos. El correo electrónico de iCloud es gratuito y cuenta con almacenamiento remoto para copias de seguridad, correo y documentos.

Todos los dispositivos, las aplicaciones y el contenido de iOS están vinculados a su ID de Apple. Cuando se enciende un dispositivo iOS por primera vez, el asistente de configuración lo guía por el proceso de conectar el dispositivo e ingresar o crear un ID de Apple. El asistente de configuración también le permite crear una cuenta de correo electrónico de iCloud. Durante el proceso de configuración, puede restaurar la configuración, el contenido y las aplicaciones de otro dispositivo iOS desde una copia de seguridad que se encuentra en iCloud.

Para configurar un grupo en el hogar, utilice la siguiente ruta de acceso:

Configuración > iCloud.

Para configurar cuentas de correo electrónico adicionales, como se muestra en la Figura 1, siga estos pasos:

Paso 1: Seleccione Configuración > Correo, Contactos, Calendarios > Agregar cuenta.

Paso 2: Pulse el tipo de cuenta, como se muestra en la Figura 2.

Paso 3: Si no figura el tipo de cuenta, toque Otro.

Paso 4: Introduzca la información de cuenta.

Paso 5: Toque Guardar.
