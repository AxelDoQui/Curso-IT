Página de prueba de la impresora

Una vez instalada la impresora, imprima una página de prueba para verificar que la impresora funcione correctamente. La página de prueba confirma que el controlador se instaló y funciona correctamente, y que la impresora se comunica con la PC.

Para imprimir una página de prueba manualmente en Windows Vista, siga los siguientes pasos:

Panel de control > Impresoras para mostrar la ventana Impresoras.

Haga clic con el botón secundario en la impresora deseada y siga esta ruta de acceso, como se muestra en la figura:

Propiedades > ficha General > Imprimir página de prueba

Un cuadro de diálogo pregunta si la página se imprimió de manera correcta. Si la página no se imprimió, los archivos de ayuda incorporados le ayudarán a resolver el problema.

Para imprimir una página de prueba manualmente en Windows 7, siga los siguientes pasos:

Dispositivos e impresoras, para mostrar el panel de control de Dispositivos e impresoras.

Haga clic con el botón secundario en la impresora deseada y siga esta ruta de acceso:

Propiedades de impresora > ficha General > Imprimir página de prueba

Para imprimir una página de prueba manualmente en Windows 8.0 o Windows 8.1, siga la siguiente ruta de acceso:

Panel de control > Dispositivos e impresoras para mostrar el panel de control de Dispositivos e impresoras.

Haga clic con el botón secundario en la impresora deseada y siga esta ruta de acceso:

Propiedades de impresora > ficha General > Imprimir página de prueba

Impresión desde una aplicación

Puede probar la impresora imprimiendo una página de prueba desde una aplicación, como el Bloc de notas o WordPad. Para acceder al bloc de notas de Windows 7 y Vista, utilice la siguiente ruta de acceso:

Inicio > Todos los programas > Accesorios > Bloc de notas

Para abrir el bloc de notas de Windows 8.0 y 8.1, desde la Pantalla de inicio, escriba bloc de notas y haga clic en bloc de notas para abrirlo.

Escriba algún texto en el documento nuevo que se abre. Para imprimir el documento, siga los siguientes pasos:

Archivo > Imprimir

Puesta a prueba de la impresora desde el panel de la impresora

La mayoría de las impresoras tienen un panel frontal con controles que permiten al usuario generar páginas de prueba. Este método de impresión permite verificar el funcionamiento de la impresora aislada de la red o la PC. Consulte el sitio web o los documentos del fabricante de la impresora para obtener información sobre cómo imprimir una página de prueba desde el panel frontal de la impresora.
