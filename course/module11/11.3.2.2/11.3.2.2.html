Servidores de impresión de software

En el tema anterior, aprendió cómo una computadora con Windows puede compartir una impresora conectada con otros equipos Windows en la red. ¿Qué ocurriría si la computadora que comparte la impresora está ejecutando un sistema operativo diferente, como Mac OS X? En una situación así, puede usar el software de servidor de impresión.

Un ejemplo es la impresora gratuita Server Bonjour de Apple, que es un servicio integrado en Mac OS X. Se instala automáticamente en una computadora Windows si instala el navegador Apple Safari. También puede descargar la impresora Server Bonjour para Windows de forma gratuita en el sitio web de Apple.

Una vez descargada e instalada, la impresora Server Bonjour opera en segundo plano, detectando automáticamente cualquier impresora compatible conectada a la red. Abra el asistente de impresora Wizard Bonjour para configurar el equipo Windows para utilizar una impresora, como se muestra en la figura.

Compartir una impresora desde una PC también tiene desventajas. La PC que comparte la impresora usa sus propios recursos para administrar los trabajos de impresión que se envían a la impresora. Si el usuario de la computadora de escritorio está trabajando al mismo tiempo que otro usuario de la red envía un trabajo para imprimir, es posible que el usuario de la computadora de escritorio advierta una disminución del rendimiento. Además, la impresora deja de estar disponible para los otros usuarios si el usuario de la PC que comparte la impresora reinicia o apaga la PC.
