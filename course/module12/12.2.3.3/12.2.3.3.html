Copias de seguridad de datos

Los datos pueden perderse o dañarse en circunstancias tales como robos, fallas de equipos o un desastre. Por este motivo, es importante realizar periódicamente copias de seguridad de datos.

Una copia de seguridad de datos almacena una copia de la información de una PC en medios de copia de seguridad extraíbles que pueden conservarse en lugares seguros. La realización de copias de seguridad de datos es uno de los métodos más eficaces para evitar la pérdida de estos. Si el hardware informático presenta una falla, pueden restaurarse los datos de la copia de seguridad en el hardware que funciona.

Las copias de seguridad de datos deben realizarse con regularidad e incluirse en un plan de seguridad. Las copias de seguridad de datos suelen almacenarse externamente, para proteger los medios de copia de seguridad en caso de que ocurra algo en la instalación principal. Los medios de copia de seguridad suelen reutilizarse para ahorrar en los costos de medios. Siempre siga las pautas de rotación de medios de la organización.

Las siguientes son algunas consideraciones para las copias de seguridad de datos:

  • Frecuencia: La realización de copias de seguridad puede llevar mucho tiempo. En ocasiones, es más sencillo realizar una copia de seguridad completa mensual o semanalmente y, luego, realizar copias de seguridad parciales de los datos que se hayan modificado desde la última copia de seguridad completa. No obstante, tener muchas copias de seguridad parciales incrementa el tiempo necesario para restaurar los datos.

  • Almacenamiento: Para obtener seguridad adicional, las copias de seguridad deben trasladarse diariamente, semanalmente o mensualmente a una ubicación de almacenamiento externa, según lo exija la directiva de seguridad.

  • Seguridad: Las copias de seguridad pueden protegerse con contraseñas. Se debe ingresar la contraseña para poder restablecer los datos de los medios de copia de seguridad.

  • Validación: Siempre deben validarse las copias de seguridad para asegurar la integridad de los datos.

Para realizar una copia de seguridad en Windows 7 o Vista, utilice la siguiente ruta de acceso como se muestra en la Figura 1:

Panel de control > Copias de seguridad y restauración

Desde aquí, puede realizar una copia de seguridad de un disco duro en un disco extraíble, crear una imagen del sistema, o crear un disco de reparación del sistema.

En Windows 8.0 y 8.1, el elemento Panel de Control Copias de seguridad y restauración se ha eliminado. En cambio, se utiliza el Historial del archivo para realizar una copia de seguridad de los archivos. El Historial del archivo se encuentra en el Panel de control. En la Figura 2, no se encontró ninguna unidad utilizable para finalizar la copia de seguridad. Primero, deberá configurar una unidad de historial de archivos y activar el historial del archivo. Puede elegir una unidad externa conectada o una unidad interna haciendo clic en Seleccionar unidad en el panel de la izquierda.
