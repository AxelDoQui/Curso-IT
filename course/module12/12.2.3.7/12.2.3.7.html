Eliminación permanente de datos

Los datos de protección también incluyen la eliminación de los archivos de dispositivos de almacenamiento cuando ya no se necesitan. La simple eliminación de archivos o formatear la unidad no es suficiente para proteger su privacidad. Por ejemplo, la eliminación de archivos de un disco duro no los elimina por completo. El sistema operativo solamente elimina la referencia al archivo en la tabla de ubicación del archivo. Sin embargo, todavía siguen los datos en la unidad. Estos datos no se eliminan por completo hasta que el disco duro almacena otros datos en la misma ubicación y sobrescribe los datos anteriores.

Por este motivo, las herramientas de software pueden utilizarse para recuperar carpetas, archivos, e incluso particiones enteras. Esto podría ser una bendición si el proceso de borrado fue accidental. Pero también podría ser desastroso si los datos son recuperados por un usuario malintencionado.

Por este motivo, los medios de almacenamiento se deben borrar completamente mediante uno o más de los siguientes métodos:

  • Datos que permiten limpiar el software: También conocida como eliminación segura, consta de una herramienta de software diseñada específicamente para sobrescribir los datos existentes varias veces, lo que hace que los datos sean ilegibles.

  • Vara desmagnetizadora: (Figura 1) Está formada por una barra con potentes imanes que se sostiene por discos establecidos del disco duro para interrumpir o eliminar el campo magnético en un disco duro.

  • Dispositivo desmagnetizador electromagnético: Compuesto por un imán con una corriente eléctrica aplicada a ella para crear un campo magnético muy fuerte. El dispositivo puede interrumpir o eliminar rápidamente el campo magnético en un disco duro.

Estos métodos se describen más detalladamente en la Figura 2.

Nota: Es importante recordar que la eliminación permanente de datos es irreversible y que los datos no se podrán volver a recuperar jamás.

Las unidades de estado sólido (SSD, solid state drives) están compuestas por memoria flash en lugar de platos magnéticos. Las técnicas frecuentes para borrar datos, como la desmagnetización, no son eficaces. Para asegurar completamente que no se puedan recuperar los datos de una SSD, realice una eliminación segura. Esto también se aplica a SSD de híbrida.

Otros medios de almacenamiento y documentos (p. ej., discos ópticos, eMMC, barras USB) también se deben destruir. Utilice una trituradora o un incinerador diseñado para destruir documentos y cada tipo de medios. Para los documentos confidenciales que deben guardarse, por ejemplo aquellos con información secreta o contraseñas, siempre manténgalos bloqueados en una ubicación segura.

Al considerar sobre los dispositivos que deban ser eliminados o destruidos, recuerde que los dispositivos además de las computadoras y los dispositivos móviles almacenan datos. Las impresoras y los dispositivos multifunción también pueden contener un disco duro que tiene mantiene en caché los documentos que se imprimieron o analizaron. Esta función en caché se puede desactivar a veces, o el dispositivo necesita ser limpiado regularmente para garantizar la privacidad de los datos. Una buena práctica de seguridad es configurar la autenticación de usuario en el dispositivo, si es posible, para evitar que una persona no autorizada cambie las configuraciones que se refieren a la privacidad.
