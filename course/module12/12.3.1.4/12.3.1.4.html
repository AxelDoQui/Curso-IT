Firewall de Windows

Un firewall deniega selectivamente el tráfico a una PC o a un segmento de red. Los firewalls suelen actuar abriendo y cerrando los puertos que utilizan diversas aplicaciones. Al abrir solo los puertos requeridos en un firewall, se implementa una política de seguridad restrictiva. Se rechaza todo paquete que no esté explícitamente permitido. En cambio, una política de seguridad permisiva permite el acceso a través de todos los puertos, excepto aquellos que estén explícitamente denegados. En el pasado, el software y el hardware venían con una configuración permisiva. Como los usuarios no tenían la precaución de configurar los equipos, la configuración permisiva predeterminada dejaba muchos dispositivos expuestos a atacantes. Actualmente, si bien la mayoría de los dispositivos vienen con una configuración altamente restrictiva, permiten una fácil configuración.

La configuración del Firewall de Windows se puede realizar de dos formas:

  • Automáticamente: Se solicita al usuario que indique qué desea hacer con las peticiones no solicitadas, es decir, Continuar bloqueo, Desbloquear o Preguntarme más tarde. Estas solicitudes pueden ser de aplicaciones legítimas que no se configuraron previamente o de un virus o gusano que infectó el sistema.

  • Administrar configuración de seguridad: El usuario agrega manualmente el programa o los puertos necesarios para las aplicaciones en uso en la red.

Para permitir el acceso de un programa a través del Firewall de Windows en Windows 7, utilice la siguiente ruta:

Panel de control > Firewall de Windows > Permitir un programa o una función a través del Firewall de Windows > Cambiar configuración > Permitir otro programa...

Para permitir el acceso de un programa a través del Firewall de Windows en Windows Vista, utilice la siguiente ruta:

Panel de control > Centro de seguridad > Firewall de Windows > Cambiar configuración > Continuar > Excepciones > Agregar programa

Para permitir el acceso de un programa a través del Firewall de Windows en Windows 8.1 y 8.0, utilice la siguiente ruta de acceso:

Panel de control > Firewall de Windows > Permitir una aplicación o la función a través del Firewall de Windows > Cambiar configuración > Permitir otra aplicación…

Si desea utilizar otro firewall de software, deberá deshabilitar el Firewall de Windows. Para deshabilitar el Firewall de Windows en Windows 7, utilice la siguiente ruta:

Panel de control > Firewall de Windows > Activar o desactivar Firewall de Windows > Desactivar Firewall de Windows (no recomendado) > Aceptar

Para deshabilitar el Firewall de Windows en Windows Vista, utilice la siguiente ruta:

Panel de control > Centro de seguridad > Firewall de Windows > Activar o desactivar Firewall de Windows > Continuar > Desactivado (no recomendado) > Aceptar

Para deshabilitar al Firewall de Windows en Windows 8.1 y 8.0, como se muestra en la figura, utilice la siguiente ruta de acceso:

Panel de control > Firewall de Windows > Activar o desactivar Firewall de Windows > Desactivar Firewall de Windows (no recomendado) > Aceptar
