Filtrado ActiveX

Al explorar la Web, es posible que algunas páginas no funcionen correctamente a menos que instale un control ActiveX. Algunos controles ActiveX están creados por terceros y pueden ser malintencionados. El filtrado de ActiveX permite explorar la Web sin ejecutar controles ActiveX.

Una vez que se instala un control ActiveX para un sitio web, se ejecuta también en otros sitios. Esto puede reducir el rendimiento o introducir riesgos de seguridad. Cuando se habilita el filtrado ActiveX, puede elegir a qué sitios web se les permite ejecutar controles ActiveX. Los sitios que no estén aprobados no pueden ejecutar estos controles, y el navegador no muestra notificaciones para que los instale o los habilite.

Para habilitar el filtro Active X en Internet Explorer 11, utilice la siguiente ruta de acceso:

Herramientas > Filtrado ActiveX

El ejemplo de la ilustración muestra que el filtrado de ActiveX está habilitado. Al hacer nuevamente clic en el Filtrado de ActiveX este se desactiva.

Para ver el contenido ActiveX en un sitio web con el filtrado de ActiveX habilitado, haga clic en el icono azul de Filtrado de ActiveX en la barra de direcciones y, luego, haga clic en Desactivar el filtrado de ActiveX.

Después de ver el contenido, puede volver a activar el filtrado ActiveX para el sitio web siguiendo los mismos pasos.
