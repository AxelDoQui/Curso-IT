Filtro SmartScreen

Los navegadores web también pueden ofrecer funcionalidades de filtrado web adicional. Por ejemplo, Internet Explorer 11 proporciona la función de filtro SmartScreen. Esta característica detecta sitios web de suplantación de identidad, analiza las páginas web para los elementos sospechosos, y revisa descargas contra una lista que contiene los sitios y los archivos conocidos por ser maliciosos.

Para activar la función de filtro SmartScreen de Internet Explorer 11, utilice la siguiente ruta de acceso como se muestra en la Figura 1:

Herramientas > Filtro SmartScreen > Activar el filtro SmartScreen

Esto abre la ventana del filtro de Microsoft SmartScreen como se muestra en la Figura 2. Desde esta ventana, el usuario puede habilitar o deshabilitar la función.

Para activar la función, haga clic en Activar filtro SmartScreen (recomendado) y haga clic en Aceptar.

El contenido de un sitio web también puede verificarse. Para analizar la página web actual, utilice la siguiente ruta:

Herramientas > Filtro SmartScreen > Comprobar este sitio web

Para informar sobre una página web sospechosa, utilice la siguiente ruta:

Herramientas > Filtro SmartScreen > Notificar sitio web no seguro
